## 1. What is a shell?
A shell is a command-line interpreter that allows users to interact with the Linux operating system.

## 2. What is shell scripting?
Shell scripting is writing a sequence of Linux commands in a file to automate tasks.

## 3. Which shell is default in Linux?
bash

## 4. What is shebang (#!/bin/bash)?
It tells the system which interpreter to use to run the script.

## 5. How do you make a script executable?
chmod +x script.sh

## 6. Difference between sh and bash
sh → basic shell

bash → advanced shell with more features

## 7. What are positional parameters?
Arguments passed to a script ($1, $2, etc.).

## 8. Explain $0, $1, $#, $@, $*
- $0 → script name

- $1 → first argument

- $# → total arguments

- $@ → all arguments (separately)

- $* → all arguments (single string)

## 9. Difference between echo and printf
- echo → simple output

- printf → formatted output

## 10. How do you take user input?
read name




## 1.What is a Function in Shell Scripting?
A function in shell scripting is a reusable block of code that performs a specific task.
</br>
Instead of writing the same commands again and again, we put them inside a function and call it whenever needed.
